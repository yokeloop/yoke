# Добро пожаловать в sp

**sp** — маркетплейс скиллов и команд для Claude Code, вдохновлённый [obra/superpowers](https://github.com/obra/superpowers).

## Структура маркетплейса

```
.claude-plugin/
  plugin.json          # манифест плагина (имя, версия, автор)
  marketplace.json     # реестр доступных плагинов
skills/                # скиллы — автоматически обнаруживаются
  <имя-скилла>/
    SKILL.md           # определение скилла (YAML frontmatter + инструкции)
commands/              # slash-команды (.md с YAML frontmatter)
```

## Как это работает

- **Скиллы** (`skills/`) активируются автоматически, когда Claude Code определяет, что задача соответствует описанию скилла. Каждый скилл — это директория с файлом `SKILL.md`.
- **Команды** (`commands/`) — это slash-команды, которые пользователь вызывает явно (например, `/hello`). Каждая команда — файл `.md` в директории `commands/`.
- **Манифест** (`.claude-plugin/plugin.json`) содержит метаданные плагина и обеспечивает его обнаружение системой Claude Code.

## Планируемые скиллы

| Команда     | Назначение              |
| ----------- | ----------------------- |
| `/task`     | Управление задачами     |
| `/brain`    | Брейнсторминг           |
| `/plan`     | Планирование реализации |
| `/do`       | Выполнение задач        |
| `/polish`   | Полировка кода          |
| `/pr`       | Создание pull request   |
| `/review`   | Код-ревью               |
| `/qa`       | Тестирование            |
| `/fix`      | Исправление багов       |
| `/memorize` | Сохранение в память     |
| `/merge`    | Мерж веток              |
